# Smart Health & Fitness Website

A demo website for a smart health app featuring smartwatch sync, AI-powered food recognition, social groups, ephemeral meal snaps, and health analytics.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development site:

```bash
npm run dev
```

3. Open the local URL shown in the terminal.

## Features

- Mock smartwatch sync button for activity updates
- Camera-based meal logging with AI food recognition demo
- Health score and energy balance calculations
- Friends list, group challenges, and disappearing meal snaps
- Responsive landing page built with React + Vite
